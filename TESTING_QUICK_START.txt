💡 TIPS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. DevTools Console Magic
   Paste this to monitor all trackEvent calls:
   
   window.originalFetch = window.fetch;
   window.fetch = function(...args) {
     if (args[0].includes('track') || args[0].includes('event'))
       console.log('EVENT:', args[0], args[1]);
     return window.originalFetch(...args);
   };

2. Database Verification
   psql "$NEON_CONNECTION_STRING" -c "SELECT * FROM analytics_events ORDER BY created_at DESC LIMIT 5"
   
3. Feature Flag Toggle (backend)
   psql "$DB" -c "UPDATE feature_flags SET enabled=true WHERE flag_name='CLUSTERS_VISIBLE'"
   
4. Screenshots
   Save any error screens with: Cmd+Shift+5 (Mac) or Win+Shift+S (Windows)
   Name them: T4.2.2_checkout_error.png (test ID + description)
