# Council Inquiry Index

**Purpose:** Single register of active and historical council inquiries.  
**Owner:** Council facilitator  
**Update cadence:** Update when an inquiry is created, changes status, or is closed.

## Status definitions

1. `draft` — authored but not ready for council review
2. `review` — ready for council discussion
3. `approved` — decision made; implementation may proceed
4. `deferred` — useful, but intentionally postponed
5. `rejected` — reviewed and declined
6. `superseded` — replaced by a newer inquiry or decision

## Operating notes

1. Every inquiry must link to a single primary artifact in `inquiries/`.
2. Supporting docs may live elsewhere, but the council reviews the inquiry artifact here.
3. When an inquiry is decided, append the outcome to `DECISIONS.md`.
